# open-data-scavenger-kt
Using Kotlin to access some open data

I'm using this to have a reason for looking at Kotlin, and using java 25 and gradle 9.6x since they are current versions when I start this (2026).
Im using smhi for weather since Im based in sweden, and aiming to look at nasas DNS and solar wind stuff and fligths.

Feel free to fork this and modify for things in line of your interests (or port to mac or android or iphone or some gui rather than textbased).
